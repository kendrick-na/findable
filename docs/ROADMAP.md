## 🚀 v1.5 (2026.11~2026.12) — Workflow 플랫폼 확장

> 🔍 **[Day02 재검토 제안 — 미확정]** (docs/day-02_시장분석 §6-4)
> **로드맵 긴장**: Findable의 핵심 차별점이 "측정"이 아니라 "개선(액션)"인데, 액션 기능 전체를 v1.5로 미룸.
> 경쟁사도 다 "측정"은 함 → v1이 측정만이면 차별화가 약함. **최소 "주간 액션 카드 1개"(읽기 전용 추천)를 v1에 넣는 것 검토** — CMS 발행 등 무거운 건 v1.5 유지하되, "이번 주 뭘 할지" 제안만이라도 v1에.
> ⚠️ 개발 범위·일정 영향 큼 → 팀 논의 후 결정. 지금은 제안만.

### 목표
**"모니터링 SaaS → 워크플로 플랫폼"**

Profound 한계 극복: 30-day 테스트에서 -1% answer share, -2.0x ROI (모니터링만으로는 성과 부족)
→ AirOps 패턴 (Greylock $40M) 차용: Insights → Actions → Operations 3-pillar
